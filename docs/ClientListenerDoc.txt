ChatListener Documentation
================================

Command Structure
-----------------

Takes the form:	(CAPS = mandatory)
TIMESTAMP COMMAND args...

Possible commands are:
- LIST = FIRST ARG IS [ENV, ROL]
	 ENV = IF ENV RETURN THE ENVIRONMENT LABEL AND THE ROLES CONTAINED
	 ROL = RETURN THE REPOS OF THE ROLE(S) IN THE NEXT ARGS
- PULL = A COMMAND TO PULL THE REPOS OF THE CURRENT ENVIRONMENT ON THE CLIENT
- CURRSTATE = RETURNS IF CONSIDERED A TEST OR DEPLOY SERVER
- STATECHANGE = CHANGE TO STATE OF NEXT ARG [TEST, DEPLOY]
- FILE = RETURN AN IMG OF THE CURRENT ENVIRONMENT
